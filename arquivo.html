Arquivos

Criar e Escrever em um Arquivo Explicação:
Criar e Escrever em um Arquivo:
with open(nome_arquivo, 'w') as arquivo: abre um arquivo para escrita ('w'), criando-o se não existir ou sobrescrevendo-o se já existir. arquivo.write(conteudo) escreve o conteúdo no arquivo. O with garante que o arquivo seja fechado automaticamente após a operação.

Ler um Arquivo Ler um Arquivo:
with open(nome_arquivo, 'r') as arquivo: abre um arquivo para leitura ('r'). conteudo = arquivo.read() lê todo o conteúdo do arquivo.
O conteúdo é retornado e impresso.

Atualizar um Arquivo Adicionar Conteúdo a um Arquivo:
with open(nome_arquivo, 'a') as arquivo: abre um arquivo para adicionar conteúdo ('a'). arquivo.write(conteudo_adicional) adiciona o novo conteúdo ao final do arquivo.

Apagar um Arquivo Explicação:
Importar o Módulo os: import os importa o módulo que fornece funções para interagir com o sistema operacional.
Função apagar_arquivo(nome_arquivo): os.remove(nome_arquivo) tenta remover o arquivo especificado pelo nome_arquivo.
Tratamento de Exceções: FileNotFoundError: Captura o erro se o arquivo não existir.
PermissionError: Captura o erro se o programa não tiver permissão para apagar o arquivo.
Exception as e: Captura outros erros e imprime a mensagem de erro correspondente.
Exemplo de Uso:
Define o nome do arquivo a ser apagado e chama a função apagar_arquivo().
